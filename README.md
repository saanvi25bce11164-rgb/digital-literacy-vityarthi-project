# digital-literacy-vityarthi-project
This repository contains my Digital Literacy Portfolio for CSE0001, covering essential skills like digital tools usage, safe internet practices, professional communication, and cybersecurity awareness.
🔹 Task 1: Digital Literacy Infographic

Created an infographic using tools like Canva/Prezi to explain digital literacy concepts such as safe internet usage, digital tools, and online professionalism.

🔹 Task 2: Student Digital Portfolio

Developed professional profiles on platforms like GitHub, LinkedIn, and others to build an online presence. Screenshots and reflections are included.

🔹 Task 3: Coding & Collaboration Platforms
Completed a beginner-level coding challenge on platforms like HackerRank/CodeChef.
Created a Google Form quiz for digital literacy awareness.

🔹 Task 4: Email Etiquette & Communication
Drafted two professional emails (to a professor and internship coordinator).
Created a checklist for responsible social media usage.
🔹 Task 5: Cybercrime Awareness
Developed a case study on a cybercrime topic (e.g., phishing, UPI fraud).
Created a prevention checklist with actionable safety tips.
